# Changelog

История изменений кабинета и админки. Формат: `## vX.Y` + пункты.
Пишем на языке пользователя (что исправили / что добавили), без внутренней кухни.

## v0.8
- Тарифы в кабинете: карточки свёрнуты в компактный список, раскрываются по клику — подсветка теперь у выбранного тарифа, а не всегда у одного и того же
- Меню бота в админке: понятная группировка (основные кнопки бота отдельно от дополнительных), редактор текста/эмодзи/цвета сворачивается по клику, а не занимает всю страницу
- Приложение Happ: ссылка подключения теперь защищённая (домен подписки не виден напрямую) — устойчивее к блокировкам
- Исправлено: кнопка «Подключиться» могла привести к ошибке и белому экрану, если открыть кабинет через мини-приложение Telegram
- Обновление через `./update.sh` стало надёжнее — устранена случайная причина ошибок сборки после ручных правок на сервере

## v0.7
- Гранулярные роли доступа в админке: выдача прав по разделам, готовые пресеты (модератор, маркетолог, поддержка), режим «только просмотр» и срок действия доступа
- Лента обновлений: раздел с историей версий прямо в админке
- Цвета кнопок меню бота настраиваются из кабинета (синяя / зелёная / красная)

## v0.6.7
- Аналитика продаж за 30 / 60 / 90 дней на дашборде
- Трафик по нодам для каждого пользователя в его карточке
- Пробный период — бесплатный, с понятной карточкой оформления
- Инфо-блок «откуда берутся данные» на странице RemnaWave

## v0.6.5
- Раздельный фон для тёмной и светлой темы
- Возможность добавлять свои приложения для подключения в кабинете

## v0.6.3
- Фильтры и сортировка в списках пользователей и транзакций
- Отслеживание входов: последний вход, IP-адрес, история
- Роль «только просмотр» для админов
- Исправлены баги в тарифах и объединении подписок
